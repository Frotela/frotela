Introduction to CSS

CSS stands for Cascading Style sheets. It is used to give styles like colour, padding, margin, fonts, font-size, alignment and many more. Which makes a web page look beautiful. We use Css in Web development. Lets see the syntax. This is how a simple css code looks like. Here p is selector, color is parameter and blue is value Selector A can be a html tag or attributes like id class Parameter In css there are different types of parameters like colour ,background colour,font size and many more Value This we will assign to that parameters according to our requirement like for color we can give blue black there are plenty of colours to use.

Types of Selectors we can use
1)id
2)class
3)element
4)element,element
5)*
6)element.class

Different ways to write Css
1)InLine
2)Internal
3)External
